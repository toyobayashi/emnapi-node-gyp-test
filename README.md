# Linux & macOS

```bash
npm install

npm run build-emscripten:unix
npm run build-emscripten-threads:unix
npm run build-wasi:unix
npm run build-wasi-threads:unix
npm run build-wasm32:unix
npm run build-wasm32-threads:unix

npm start
```
